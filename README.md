# 🎵 VibePlay - Music Player

A modern Spotify-inspired music player built using HTML, CSS, and JavaScript. This project provides a clean and responsive user interface with essential music playback features.

## 🚀 Features

* Play and Pause Music
* Next and Previous Track Navigation
* Interactive Playlist
* Progress Bar with Seeking
* Volume Control
* Responsive Design
* Spotify-Inspired User Interface
* Automatic Track Switching
* Smooth User Experience

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 🎯 How to Run

1. Download or clone the repository.
2. Add your music files to the songs folder.
3. Open index.html in your browser.
4. Enjoy listening to your favorite songs.
